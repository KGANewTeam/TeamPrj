## 📚 Rules & Guidelines

<details>
<summary><b>📁 에셋 관리 규칙</b></summary>
<div align="center">

### ⚙️ 에셋 관리 규칙
━━━━━━━━━━━━━━━━━━━━━━

#### • 외부 에셋 설치
  구글 드라이브의 External 압축파일을 Asset 폴더 내 설치  
  에셋 스토어 패키지는 반드시 팀장과 상의 후 설치  

━━━━━━━━━━━━━━━━━━━━━━

#### • 신규 에셋 추가
  External 폴더에 임포트 후 압축하여 드라이브 업로드  
  파일명: `External_MMDD_HHMM` (예: External_1227_1800)  
  추가된 에셋 정보를 팀 디스코드에 공유  

━━━━━━━━━━━━━━━━━━━━━━

#### • 에셋 네이밍 규칙
  영문 사용 (한글 사용 금지)  
  띄어쓰기 대신 카멜케이스 사용  
  프리팹: `Pref_기능명`  
  머티리얼: `Mat_용도명`  
  텍스처: `Tex_용도명`  

━━━━━━━━━━━━━━━━━━━━━━

</div>
</details>
